# ju-scooter

## About The Project

The JU-Scooter project aims to create a modern, campus-based electric scooter rental service designed to address the unique transportation needs of University of Jordan students.
The app will provide a convenient, eco-friendly alternative to existing bus services, enabling students to travel around campus efficiently and sustainably.
